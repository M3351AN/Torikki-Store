# Torikki-Store
KernelSU module to fill target.txt of Tricky Store with Web UI. Also auto check if keybox valid. Your KernelSU Manager must support Web UI.

## Changelog
### Shimo tsuki (1003)

- Removed method that read KSU allowlist
- Added Keybox checker
